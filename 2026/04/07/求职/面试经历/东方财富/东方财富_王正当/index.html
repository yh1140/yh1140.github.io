第三部分：技术深度问答

1. 开发能力与编程语言

面试官： 现在的运维很多属于“运开”（运维开发），对开发是有要求的，至少要能看懂代码。你目前最熟悉的开发语言是什么？

王正当： 我最熟悉 Python。在校期间做过两个 Web 项目（APK 研判系统和 AI 辅助审片），用的都是 Python 的 Django 框架。我对自学能力很有信心，当时项目需要的 Web 前端（HTML/CSS）、Git 以及深度学习知识很多都是自学的。在项目中我负责整体后端的接口编写和模型微调。

2. 算法基础：冒泡排序

面试官： 你能描述一下“冒泡排序”吗？

王正当： 冒泡排序是一种经典的交换排序。它通过重复遍历序列，比较相邻的两个数。如果顺序错误（比如想从小到大排，但前一个比后一个大），就交换它们。这样每一轮下来，最大（或最小）的数就会像气泡一样“浮”到序列的一端。

3. 网络协议：NAT

面试官： 讲一下什么是 NAT？

王正当： NAT 是网络地址转换协议。它在数据包传输过程中改变原 IP 或目的 IP。

  • SNAT（源地址转换）： 常用于局域网共享上网，将内网私有 IP 转换为公网 IP。

  • DNAT（目的地址转换）： 常用于在外网访问内网服务器。比如我们不在公司，但要访问公司内网机器，通过公网 IP 转换进入。

4. 架构理解：微服务与项目结构

面试官： 谈谈你对服务架构的理解。如果让你画一个你做过的系统架构图，大概包含哪些部分？

王正当： 现在的趋势是微服务架构，将功能模块化。

  • 接口层： Django 处理前端请求。

  • 存储层： MySQL 存储 APK 的特征信息。

  • 处理层： APK 解析模块（提取特征）和深度学习模型（研判是否有违规、色情内容）。 这种模块化开发便于后期的维护和某个功能的独立升级，互不影响。

5. 运维工具链

面试官： 你用过哪些运维工具，分别解决什么问题？

王正当：

  • Prometheus： 核心监控系统，采集服务器指标。

  • Zabbix： 用于监控和报警。

  • Ansible： 自动化运维，批量管理和操作机器。

  • GitLab CI/CD： 负责运维自动化的流程管理。 我觉得对于新工具，只要理解了原理，上手其实很快。

男面试官： 好的。我看你用了 Prometheus 和那个 Grafana 进行了一个监控，对吧？ 王正当： 嗯。 男面试官： 你觉得 Prometheus 和那个 Zabbix 它们之间有什么差异吗？ 王正当： Prometheus 的话，它是用那个 Exporters 来拉一些就是机器上面的一个数据嘛。这个 Zabbix 的话，它的……Zabbix 它是要在每台机器上都弄一个 Zabbix Agent 来去拉取每台机器上的数据。我觉得 Prometheus 比 Zabbix 的话，就是当时在了解这两个监控软件的话是 Zabbix 它不适用于就是如果说在几百几千台服务器的情况下，Zabbix 的 Server 是可以支持一个很大的并发，但是如果说机器的一个量上去了的话就是 Zabbix 它就满足不了高并发，就是高集群的一个这样的一条件。然后 Prometheus 我觉得它还比较好的一个就是它的……它的一个采集器嘛，它能够采集很多的数据。就是现在有很多……对它开源的像它不仅能有 Node Exporter 采集你的主机数据，而且就是关于其他的进程……采集你的容器的一些指标。而且专门对于一些应用 App，比如说 Nginx Exporter，还有 MySQL Exporter 能够采集特定……特定应用的一个 Exporters 采集器。我觉得就会比较方便一点。

男面试官： 嗯。我听你说你有两项软件著作权对吧？那个是什么软件著作权？ 王正当： 是在学校里面做的一些项目开发经历。一个是关于……就是 APK 研判系统。当时是这两项软件著作权都是 Web 系统的嘛，就做了一个 Web。然后一个是关于 APK 研判的。一个是关于使用 AI 辅助审片系统的。简单来说第一个就是……你对 APK 就是我们所说的应用程序做逆向解析。然后逆向解析的目的是为了区别市面上的一些黑灰产和一些正常可用一个 APP 的一个做一个智能研判。是这两个其实都是用的机器学习和深度学习的一个方法。第二个 AI 辅助审片的话是主要通过计算机视觉来快速地框选出你比如说电影……电影镜头里面那种有吸烟场景的镜头或者说一些色情暴力的镜头，对他进行一个框选，并且把它的时长进行一个统计。

男面试官： 对。不好意思，刚刚第二个说是对电影的一些镜头进行框选，进行统计？这个是通过什么方式来实现的？ 王正当： 就……什么方式？是计算机视觉里面通过深度学习，通过深度卷积。就是对一张图片，比如说简单来说就是一张图片是否是……是否是小狗还是猫，做一个简单的分类嘛。然后这里的分类就是是否有吸烟镜头或者说一些暴力镜头。然后的话这里面还加入了……加入了目标检测。目标检测的话它就会……就是框选出比如说一张动物的话，我目标检测我能够框出这张动物的它的一个坐标，就是用一个矩形框把它给框出来。然后这里面的话就是把这个吸烟的镜头给它框出来。 男面试官： 这个是你独立完成的？ 王正当： 就这两项……就这两项项目的话都是小组形式，就是团队完成的。但我都是这里面的一个负责人。 男面试官： 团队成员是你找的？ 王正当： 对，都是我找的。 男面试官： 是你们上课时候的课题吗？ 王正当： 第一个是比赛的一个赛题，第二个是……就第二个 AI 辅助审片是我主持的一个省级大创项目。

男面试官： 你对那个……就是金融行业的这些 IT 相关的有了解？ 王正当： 可能了解的不是很多。就是……因为如果说以后的话，如果说有机会从事的话肯定也会去多多学习一些相关的知识。 男面试官： 自己也没做过什么基金啊股票这些交易对吧？ 王正当： 可能只听说过，对。 男面试官： 行，我的问题就这些。

女面试官： 那，你投递我们公司的原因是什么？或者是通过什么途径知道它的？ 王正当： 当时就是也是海投，然后筛了一下吧。就是觉得……因为咱们公司也不错，然后就选择了一个。然后可能里面一些技术要求呀自己也大致能够符合，所以就投了。 女面试官： 不好意思我有点没听清楚。 女面试官： 现在是在学校里面吗？ 王正当： 对，是的。 女面试官： 现在在学校。然后……那个实习是因为暑假的原因然后到现在结束了对吧？ 王正当： 嗯，是的。 女面试官： 你有什么爱好吗？ 王正当： 平时的话喜欢去外面旅游旅游或者说还就健身健身吧。 女面试官： 工作了以后可能就是可以旅游的时间会减少，假期因为相对来说会比较少，那……那旅游会怎么安排呢？ 王正当： 这个的话就是……如果说想让自己，肯定是需要放松的嘛。因为不能一直让自己处于一个高压的情况。可能会就是找时间如果说有空的话还是希望能出去走一走。如果说没空的话会选择一些其他的兴趣爱好去听听歌或者说去弹弹吉他或者去学习一些乐器，学习一些新东西来给自己的生活添加一些色彩吧。

女面试官： 会期待那个即将入职的职场生活环境工作是一个怎么样的一个期待环境？ 王正当： 工作环境嘛。工作环境肯定是希望能积极向上一点吧。然后……彼此同事之间能够友好和善。然后也是就是有一个比较良好的积极向上的氛围吧。就是学习为主，然后大家都抱着一个把公司为了公司去做一些更好的就是发展为目标。

女面试官： 你爸妈对于你就业有没有什么指导意见之类的？包括城市呀、岗位呀、或者之类的一些指导建议。 王正当： 就是他们可能就是更尊重我自己的意见吧。而我自己的话可能就是对于喜欢……可能更倾向于去东部呀、东部那边或者是……因为我家是湖南的嘛，我可能会更倾向于去东部或者说是南边的广东深圳那边。然后对于工作的一个具体城市的话肯定也是希望在一个像北上广深这样机会多一点的城市。

女面试官： 好。那最后你还有什么想要了解的吗？ 王正当： 就是想问一下，就是咱们公司对于应届生的一些入职……就是一些评判与要求。在招聘上有什么是能够加分的点，或者是更看重应届生哪一些……哪一些特点吧。

女面试官： 面试官能回答一下这个问题吗？对于这个岗位有哪些点是比较看重的、是比较加分的。 男面试官： 这个岗位呢，实际上是这样子的。应届生呢肯定都是一样的，我们都知道应届生的实际的应用场景或者是经验相对来说比较欠缺一点。但是呢，就看他的学习成长能力，自己对技术感不感兴趣。就是从技术角度上来说是很重要的一件事情。另一个方向呢就是业务方面。业务毕竟是一家期货公司，对于一家期货公司这样的金融机构来说呢，业务是必不可少的部分。当你涉及会架构一些网络搭建、一些系统的时候，你首先得了解一下业务结构是怎么样的一个关系。像你刚刚系统里面那个 Web 集群的高可用模型对吧？那它其实也有它的应用场景。因为首先它得有……那么庞大的用户、大量的一个并发访问的时候，它才会有起那些作用。其实包括系统架构、金融科技里面的一些 IT 上面的这些东西呢也是这样。所以对于哦应届生或者是说在招聘的时候更关注的就是这两点：一个是学习能力，一个是对于业务对技术的一个感兴趣的一个状况。

王正当： 嗯好的，谢谢您的回答。 女面试官： 还有什么想要了解的吗？ 王正当： 那我这边应该没有其他的想法了……其他的问题了。 女面试官： 行。那如果没有什么其他问题的话，我们今天的面试就到这边结束。非常感谢你参加我们的面试。如果有反馈的话我应该会在下周的时候给到一个反馈。好吗？谢谢。 王正当： 好好。也谢谢二位的一个面试机会。 女面试官： 好，再见。 王正当： 好的，拜拜。
