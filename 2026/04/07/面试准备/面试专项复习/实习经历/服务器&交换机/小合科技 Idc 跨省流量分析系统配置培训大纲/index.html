小合科技 Idc 跨省流量分析系统配置培训大纲

一、培训目标

让新接触员工可以明白如何使用以及部署sflow程序，以及如何操作web系统、调整各项参数

二、培训对象

所有不清楚使用以及部署 sflow 程序的员工

三、培训时长

一周

四、培训内容与安排

（一）采集器主机部署实操

熟悉掌握采集器主机的配置，如程序的部署、程序主机如何部署（控制器部署过程）

客户提供服务器（pve）

一、PVE操作

1.1 登录pve 网页

检查一下网络模式是否为桥接模式，不是桥接模式要先改为桥接模式

登录后：立即改密码，密码10以上，大小写、数字、特殊字符组成

修改密码后：加防火墙，通过jumpserver 进行下述操作

相关防火墙：

控制器防火墙

sudo ufw allow from 119.45.6.137 to any

sudo ufw allow from 175.4.55.190 to any port 8000:9000 proto tcp

sudo ufw allow from 43.136.123.93 to any port 8000:9000 proto tcp

sudo ufw allow from 8.137.48.76 to any port 8000:9000 proto tcp

sudo ufw allow from 120.226.50.46 to any port 8000:9000 proto tcp

sudo ufw allow from 175.0.103.153 to any port 22

sudo ufw allow from  175.6.10.122 to any port 8000:9000 proto tcp

sudo ufw allow 6344/tcp

sudo ufw allow 6343/udp

sudo ufw allow 6343/tcp

sudo ufw allow 6344/udp

sudo ufw allow 6342/udp

sudo ufw allow 6342/tcp

完成jumpserve录入后删除公司IP

sudo ufw delete 规则编号

Pve防火墙

iptables -L –line-numbers -n -v #查看防火墙规则

iptables -A INPUT -s 127.0.0.1 -p all -j ACCEPT

iptables -A INPUT -s 175.6.60.199 -p all -j ACCEPT

iptables -A INPUT -s 119.45.6.137 -p all -j ACCEPT

iptables -A INPUT -s 175.4.55.190 -p all -j ACCEPT

iptables -A INPUT -s 43.136.123.93 -p all -j ACCEPT

iptables -A INPUT -s 8.137.48.76 -p all -j ACCEPT

iptables -P INPUT DROP #禁止

service iptables save

1.2  pve管理页面下载虚机镜像

1.2.1 从下载站下载

wget http://175.4.55.185:8024/VMubuntu.qcow2

1.2.2 从本地上传（只允许通过jumpserver上传）

如果本地有镜像，也可以通过jumpserver 上传

结果如下：

 

1.3  pve创建虚机，然后拆解并删除虚拟硬盘

1.3.1 创建虚机，名称写虚机ip

 

1.3.2 操作系统按“不使用任何介质”，磁盘选200G，cpu 4核，内存8192，其他默认

 

1.4  关机后，拆解并分离自动创建的虚拟硬盘、CD-ROM

 

1.5 导入QCOW2磁盘镜像文件到新创建的虚拟机

pve管理页面shell窗口（shell窗口只能如果jumpserver 打开），按镜像文件创建虚拟磁盘

根目录下执行下面命令：100 为对应的ID号

 qm importdisk 101 VMubuntu.qcow2 local-lvm

1.6 把刚创建的虚拟磁盘添加进来   

 

1.7 修改虚拟机启动顺序为硬盘优先，请勾选开机自启动

 

1.8开机启动，通过控制台进入登录页面  镜像：root zc092817

 

1.9 将上面分离的硬盘添加进去

二、控制器操作

修改控制器密码，添加防火墙规则，加入jumpserver

  然后通过jumpserver，完成下述步骤

2.1 Luks硬盘加密

使用Cryptsetup 对新硬盘进行加密

命令为：

[root@nolinux ~]# cryptsetup luksFormat /dev/sdb

  然后输入大写的YES，接着输入两遍密码即可（密码统一用Xiaohe*!402#）

 

2.2 对加密分区/dev/sdb做映射

[root@119 ~]# cryptsetup luksOpen /dev/sdb nolinux

 

2.3在解密的映射文件上创建ext4文件系统

[root@119 ~]# mkfs -t ext4 /dev/mapper/nolinux

 

2.4挂载解密后的映射设备上的文件系统

[root@nolinux ~]# mkdir /nolinux   #创建挂载点

[root@nolinux ~]# mount /dev/mapper/nolinux /nolinux/   #挂载加密分区

 

（二）交换机 sFlow 配置实操

（三）Web 界面使用指南

参考培训资料：《小合科技 Idc 跨省流量分析系统配置用户使用说明书》第七部分内容

（四）检查如何排查未进调度节点流量IP

掌握通过跨省系统的其他业务和源IP查找功能，检查故障节点的默认流量、源IP

（五）使用sflow测试机做到流量结算功能
