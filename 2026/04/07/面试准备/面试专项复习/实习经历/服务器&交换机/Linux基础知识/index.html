7.3 虚拟终端会话” 的命令行工具

#基础操作
命令 作用
screen 直接创建一个新的默认会话（进入会话后，操作与普通终端一致）
screen -S 会话名 创建一个指定名称的会话（方便后续识别，如 screen -S mytask）
screen -ls 列出所有正在运行的 screen 会话（显示会话 ID 和名称）
screen -r 会话ID/名称 附着到指定会话（重新进入该会话，如 screen -r mytask 或 screen -r 12345）
screen -d 会话ID/名称 分离指定会话（将其后台运行，不进入会话）
screen -d -r 会话ID/名称 先分离会话（若已被附着），再重新附着（解决 “会话已存在” 的冲突）

#会话内操作（快捷键）
捷键 作用
Ctrl + A + d 分离当前会话（回到原终端，会话后台运行）
Ctrl + A + c 在当前会话中创建一个新窗口（可切换窗口操作）
Ctrl + A + n 切换到下一个窗口
Ctrl + A + p 切换到上一个窗口
Ctrl + A + 窗口编号 直接切换到指定窗口（窗口编号从 0 开始）
Ctrl + A + k 关闭当前窗口（会提示确认，输入 y 终止窗口内进程）
Ctrl + A + ? 显示所有快捷键帮助

#关闭或删除会话

（1）在会话内执行 exit 命令，或按 Ctrl + D，可直接关闭当前会话（会终止会话内所有进程）

（2）若会话已分离，可通过 screen -S 会话ID/名称 -X quit 强制删除会话（需谨慎，会终止进程）
