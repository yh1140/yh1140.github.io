Linux基础_硬件信息采集

发表于2026-04-02|更新于2026-04-07|Linux操作系统
|浏览量:

在 Linux 系统中，采集硬件信息主要依靠读取 /proc 和 /sys 伪文件系统中的内核数据，或者使用专门的扫描工具。

综合硬件扫描工具

如果你需要一份详尽的硬件清单，这两个命令是首选：

  • lscpu：专门查看 CPU 架构信息（核心数、频率、缓存、虚拟化支持）。
  • lshw (List Hardware)：导出极其详细的硬件配置。
    • sudo lshw -short：以摘要形式显示。
    • sudo lshw -html > hardware.html：生成可视化报告。
  • inxi：运维界常用的全能查看工具（非系统内置，需安装），显示效果非常直观
    • inxi -Fx：查看完整的硬件详情。

核心组件分类采集

CPU信息

  • cat /proc/cpuinfo：查看主频、型号等底层细节。
  • uptime 或 top：查看当前的 CPU 负载情况。

内存RAM信息

  • free -h：最常用的查看当前内存使用量、剩余量的命令。
  • dmidecode -t memory：查看物理内存插槽信息、单条内存容量及主频（需要 sudo）。
  • cat /proc/meminfo：内核视角的内存分配详情。

磁盘与存储

  • lsblk：以树状结构查看所有块设备（硬盘、分区、挂载点）。
  • fdisk -l：查看分区表详细信息。
  • smartctl -a /dev/sda：检查硬盘的健康状态 (SMART 信息，需安装 smartmontools)。
  • df -h：查看磁盘空间占用情况。

显卡GPU信息

  • lspci | grep -i vga：查看显卡型号。
  • nvidia-smi：如果你使用 NVIDIA 显卡并安装了驱动，这是监控 GPU 状态的标准命令。
  • glxinfo | grep "OpenGL render"：查看集成显卡或当前渲染器信息。
