SHELL基础_概念

发表于2026-04-02|更新于2026-04-02|shell
|浏览量:

Shell 脚本（Shell Script）本质上是一个包含了一系列命令的文本文件。它由 Shell 解释器（如 Bash、Zsh）逐行读取并执行，是自动化运维、系统管理和重复性任务处理的核心工具。

什么是shell

Shell 是用户与 Linux 内核之间的接口（解释器）。

  • 内核 (Kernel)：负责管理硬件。
  • Shell：接收你的指令并传达给内核。
  • Bash (Bourne Again Shell)：目前 Linux 发行版中最通用的标准 Shell。

脚本的基本结构

一个标准的 Shell 脚本通常包含三个部分：

  1. Shebang (#!)：指定脚本使用的解释器。
  2. 注释：以 # 开头，解释代码逻辑。
  3. 可执行命令：具体的逻辑代码。
