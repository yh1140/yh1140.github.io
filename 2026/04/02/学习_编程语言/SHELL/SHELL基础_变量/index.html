SHELL基础_变量

发表于2026-04-02|更新于2026-04-07|shell编程语言
|浏览量:
  • 双引号内的变量可以引用
  • 单引号-所见即所得

变量-类型

  • 局部变量 / 全局变量
  • 自定义变量 / 预定义变量
  • 位置参数变量
    • $0 当前执行的脚本文件名
    • $1-9 第一个到第九个参数
    • ${10} 第十个以及后面的参数必须加花括号
    • $* 所有参数(视为整体字符串)
    • $@ 所有参数(视为独立的多个字符串)
  • 预定义特殊变量
    • $$ 当前终端的进程号
    • $? 上一条命令的退出状态码
    • $# 传递给脚本或函数的参数总个数

变量-定义

  • 变量可以不定义直接使用，值为空
  • 等号两边不能有空格
  • 命名规则：由字母、数字和下划线组成，不能以数字开头。
  • 习惯上，环境变量用大写，局部变量用小写。

变量-引用

  • 单引号-强引用，所及及所得
  • 双引号-弱引用，内部可以引用变量

变量-操作

只读变量

使用 readonly 命令，变量的值将无法被修改或删除。

删除变量

使用 unset 命令。注意：无法删除只读变量。

命令结果赋值

将命令的输出结果保存到变量中，有两种写法

变量-数学运算

由于变量默认是字符串，直接相加会变成字符串拼接。若要进行数学运算，需要特殊语法

使用 $(( )) 支持嵌套

使用 let 命令

小数运算bc

  • 基础用法-管道调用
  • scale保留小数
  • bc 默认的 scale（小数精度）为 0。如果你需要保留小数，必须显式设置 scale
  • 进制转换
  • bc 是处理二进制、八进制、十六进制转换的最强工具。它有两个关键变量
    • ibase：输入进制 (input base)
    • obase：输出进制 (output base)

变量-运算&比较

(( )) —— 数学运算与数值比较

  • 支持复杂的算术运算符：+, -, *, /, %, **（幂）。
  • 变量引用可以省略 $：在括号内部，你可以直接写变量名。
  • 支持自增/自减：如 i++ 或 ++i。
  • 比较符号更符合直觉：可以直接使用 >, <, >=, <=。

[[ ]] —— 逻辑测试与条件判断

[[ ]] 是传统 [ ]（test 命令）的增强版，主要用于判断文件属性、字符串比较和复杂的逻辑组合。

  • 防止路径展开和单词分割：内部变量不需要加双引号（即使变量有空格）。
  • 支持模式匹配和正则：可以使用 == 进行通配符匹配，或使用 =~ 进行正则表达式匹配。
  • 逻辑运算符更简洁：支持 && (与) 和 || (或)，而不需要用传统的 -a 或 -o。

[[ ]]和(())对比

逻辑运算 &&, `
变量引用 $a 或 a (可省略 $) 必须使用 $a
适用类型 仅限整数 字符串、文件、整数（需用-eq等）
