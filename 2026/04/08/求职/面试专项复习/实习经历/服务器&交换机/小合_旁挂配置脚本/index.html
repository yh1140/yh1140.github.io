发表于2026-04-08|更新于2026-04-08

第一步，查看该交换机版本 需不需要升级

dis ver

snmp-agent sys-info version v2c

改版本号

snmp-agent protocol source-status all-interface

第二步，配置用户名和密码 开启ssh 配置snmp

aaa

local-user xiaohe123 password irreversible-cipher ”密码“

 local-user xiaohe123 service-type ftp ssh

 local-user xiaohe123 level 3

 local-user xiaohe123 ftp-directory flash:

stelnet server enable

sftp server enable

ssh user xiaohe123

ssh user xiaohe123 authentication-type all

ssh user xiaohe123 service-type all

ssh user xiaohe123 sftp-directory flash:

ssh server-source all-interface

snmp-agent community read cipher  ”snmp码“  acl 2000

第三步，提高用户在线人数

user-interface maximum-vty 21

user-interface vty 0 20

 authentication-mode aaa

protocol inbound all

第四步，开启20秒采集

Set flow-stat interval 20

第五步，录入acl

acl number 2000

第9步  配置eth-trunk

interface Eth-Trunk11

trunkport 100ge 1/0/2 1/0/4

 port link-type trunk

 undo port trunk allow-pass vlan 1

 port trunk allow-pass vlan 411 611  811 911

 stp disable     ####多聚合可以开，防止环路

 mode lacp-static  ###需要双方配置

第十步，配置bfd

bfd

bfd 名字  bind peer-ip 10.164.1.1 vpn-instance juhe1 interface Vlanif411

 discriminator local 20

 discriminator remote 10

第十一步，配置回流路由

ip route-static vpn-instance juhe1 0.0.0.0 0.0.0.0 10.168.1.1 description juhe1
