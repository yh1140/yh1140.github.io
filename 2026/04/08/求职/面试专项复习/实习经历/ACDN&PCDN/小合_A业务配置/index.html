                                         A业务配置指导

一、系统镜像

http://isoredirect.centos.org/centos/7/isos/x86_64/

 

二、只留万兆光口生效，如有内置的千兆网卡，需在BIOS禁用掉。

 

三、在安装系统时，将万兆网口名称修改为eth0:

1、在安装界面移动光标到 Install Centos 7.按 TAB 键，在出现的代码的末尾添加：net.ifnames=0 biosdevname=0.按下回车开始安装即可，系统会自动从eth0开始命名

 

四、安装系统时，将sda系统盘，划分独立分区
1、独立分区挂载点为/pcdn_data/pcdn_index_data (大于50GB)
例如: 系统盘为sda，划分独立分区为/dev/sda4，则将/dev/sda4挂载到/pcdn_data/pcdn_index_data目录

五、ssd进行分区（除系统盘）

1、SSD盘进行同等大小分区，比如4T SSD盘可分为4个大小相同的1T分区

fdisk /dev/

六、格化式分区并修改文件格式为xfs

  mkfs.xfs -f /dev/sdb1

  mkfs.xfs -f /dev/sdc1

  mkfs.xfs -f /dev/sdd1

  mkfs.xfs -f /dev/sde1

  mkfs.xfs -f /dev/sdf1

  mkfs.xfs -f /dev/sdg1

  mkfs.xfs -f /dev/sdh1

 mkfs.xfs -f /dev/nvme0n1p1

 mkfs.xfs -f /dev/nvme0n1p2

 mkfs.xfs -f /dev/nvme0n1p3

 mkfs.xfs -f /dev/nvme0n1p4

 mkfs.xfs -f /dev/nvme0n1p5

 mkfs.xfs -f /dev/nvme1n1p1

 mkfs.xfs -f /dev/nvme1n1p2

 mkfs.xfs -f /dev/nvme1n1p3

 mkfs.xfs -f /dev/nvme1n1p4

 mkfs.xfs -f /dev/nvme1n1p5

七、创建挂载目录并挂载磁盘

1、在根目录，创建两个文件夹mkdir pcdn_data、 mkdir pcdn_data_hdd （ssd、hdd的父目录）

mkdir /pcdn_data

mkdir /pcdn_data_hdd

2、在父目录中（pcdn_data，pcdn_data_hdd），创建ssd、hdd子目录：storage1_ssd storage1_hdd （有几个分区，则创建几个子目录）

mkdir storage1_hdd

mkdir storage2_hdd

mkdir storage3_hdd

mkdir storage4_hdd

mkdir storage5_hdd

mkdir storage6_hdd

mkdir storage7_hdd

mkdir storage1_ssd

mkdir storage2_ssd

mkdir storage3_ssd

mkdir storage4_ssd

mkdir storage6_ssd

mkdir storage7_ssd

mkdir storage8_ssd

mkdir storage9_ssd

mkdir storage10_ssd

3、临时挂载磁盘

mount /dev/sdb1 /pcdn_data_hdd/storage1_hdd/

mount /dev/sdc1 /pcdn_data_hdd/storage2_hdd/

mount /dev/sdd1 /pcdn_data_hdd/storage3_hdd/

mount /dev/sde1 /pcdn_data_hdd/storage4_hdd/

mount /dev/sdf1 /pcdn_data_hdd/storage5_hdd/

mount /dev/sdh1 /pcdn_data_hdd/storage7_hdd/

mount /dev/nvme0n1p3 /pcdn_data/storage3_ssd/

八、关闭selinux

vi /etc/selinux/config

SELINUX=disabled

九、添加并配置/etc/pcdn/pcdn.conf（全小写）
macs: 本机mac地址
nics: eth0

十、关闭防火墙

systemctl stop firewalld

十一、配置允许访问的ip

1、yum -y install xinetd    #安装xinetd工具

2、vi /etc/hosts.allow      #进入允许访问文件，添加允许访问的IP

sshd:101.91.92.223:allow

sshd:122.192.127.45:allow

sshd:175.6.107.254:allow

3、vi /etc/hosts.deny   #进入拒绝访问文件

sshd:all:deny           #表示拒绝所有ip访问，除允许访问IP外

4、service xinetd restart   # 重启xinetd 服务

十二、主机配置SNMP监控网卡流量

1、yum -y install net-snmp net-snmp-utils  #安装SNMP服务

2、vi  /etc/snmp/snmpd.conf            #进入SNMP配置文件

3、com2sec notConfigUser  default    public改为：com2sec notConfigUser    #允许被监控ip  团体名

4、 access  notConfigGroup “”   any   noauth  exact  systemview  none none  #改为 access  notConfigGroup “”   any   noauth  exact  all  none none

5、 把这两行的#号去掉

#view all    included  .1    80

#view mib2 included  .iso.org.dod.internet.mgmt.mib-2 fc  

6、systemctl restart snmpd.service   #重启SNMP服务

7、systemctl enable snmpd.service      #SNMP服务开启自启

十三、上传系统内核文件 及安装程序包至root目录并重启

1、rpm -ivh –force kernel-5.4.119-19.0006.tl2.x86_64.rpm  #更新内核

2、reboot#重启设备

3、uname -a #查看内核是否更新成功

十四、提交验收表格

十五、在root目录下运行安装程序

1、tar -xzvf installer-0.7.23.tar.gz  #解压安装程序包

2、cd installer-amd64-0.7.23 #进入此目录

3、./edgeadm join –token   #运行安装程序
