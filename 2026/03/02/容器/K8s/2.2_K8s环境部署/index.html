1. 核心定义与版本演进

从 v1.24 版本开始，K8s 正式移除了 Dockershim。这意味着 Docker 不再是默认的容器运行时。

1.1 架构对比

  • v1.23 (旧架构)：链路较长，依赖 Docker 守护进程。
  • v1.24+ (新架构)：链路精简，直接调用 containerd，效率更高。
