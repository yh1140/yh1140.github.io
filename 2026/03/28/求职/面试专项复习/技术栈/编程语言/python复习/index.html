基础库

库名称 核心用途 你的项目关联 / 场景
os / sys 系统路径管理、环境变量读取、命令行参数解析 编写 openEuler/KylinOS 自动化环境检查脚本。
subprocess 执行 Shell 命令并获取返回结果（标准输出/错误） 在 Ansible 无法覆盖的复杂场景下，手动调用系统底层工具。
logging 记录程序运行日志，支持等级分发（INFO/ERROR） 为运维自动化工具增加审计追踪，便于排查 P0 级事故。
json / yaml 配置文件解析与数据交换格式处理 解析 K8s 的 YAML 资源清单或处理 API 返回的数据。
requests 发起 HTTP/HTTPS 请求，处理 Web API DeepSeek 项目：通过 API 调用大模型进行 Text-to-SQL 转换。

数据分析挖掘可视化

库名称 核心用途 你的项目关联 / 场景
NumPy 高性能多维数组运算、线性代数处理 交通大数据项目中对海量坐标点进行数值矩阵化处理。
Pandas 结构化数据（DataFrame）清洗、转换与统计分析 处理 MySQL 导出的 CSV 数据，进行慢查询日志的时间序列分析。
Pyecharts 生成交互式、动态的可视化图表（百度开源） 交通项目：动态渲染热点图和趋势图，生成决策报告。
Polars (2026新秀) 极速 DataFrame 库，支持并行处理 处理超大规模交通流量数据时，作为 Pandas 的性能替代方案。
Matplotlib 基础绘图库，支持各类统计图表生成 实验阶段快速绘制模型损失函数（Loss）曲线。

库名称 核心用途 你的项目关联 / 场景
PyTorch 动态计算图，深度学习模型训练与推理 在校期间进行目标检测模型或预测算法的实验开发。
Ultralytics YOLO 系列模型的训练与部署框架 交通项目：利用 YOLO 进行实时视频流中的车辆检测与跟踪。
Scikit-learn 传统机器学习算法（回归、聚类、分类） 进行交通拥堵预测时的特征工程与基准模型建立。
OpenCV 计算机视觉处理（图像读取、滤波、变换） 处理实时监控视频流，进行帧提取和预处理。
FastAPI 高性能异步 Web 框架，常用于部署 AI 模型 API 将 Text-to-SQL 功能封装成微服务供前端调用。

Paramiko 基于 SSHv2 协议的远程控制与文件传输 在未安装 Ansible 的环境下实现跨主机的远程命令执行。
Ansible-Runner 在 Python 程序中调用 Ansible 任务 实现“混合云 Web 集群”中从 Python 后端触发自动化扩容。
kubernetes K8s 官方 Python 客户端 SDK 编写 Python 脚本动态监控 Pod 状态或自动调整副本数。
SQLAlchemy 对象关系映射（ORM），支持复杂 SQL 抽象 在 Text-to-SQL 项目中，将模型生成的 SQL 语句安全地映射到库表。
