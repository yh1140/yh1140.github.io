基础库

库名称 核心用途 你的项目关联 / 场景
os / sys 系统路径管理、环境变量读取、命令行参数解析 编写 openEuler/KylinOS 自动化环境检查脚本。
subprocess 执行 Shell 命令并获取返回结果（标准输出/错误） 在 Ansible 无法覆盖的复杂场景下，手动调用系统底层工具。

json / yaml 配置文件解析与数据交换格式处理 解析 K8s 的 YAML 资源清单或处理 API 返回的数据。
requests 发起 HTTP/HTTPS 请求，处理 Web API DeepSeek 项目：通过 API 调用大模型进行 Text-to-SQL 转换。

库名称 核心用途 你的项目关联 / 场景

Pandas 结构化数据（DataFrame）清洗、转换与统计分析 处理 MySQL 导出的 CSV 数据，进行慢查询日志的时间序列分析。

Matplotlib 基础绘图库，支持各类统计图表生成 实验阶段快速绘制模型损失函数（Loss）曲线。
