Redis概念_基本概念
Redis（Remote Dictionary Server）是一个开源的、基于内存的数据结构存储系统。它通常被用作数据库、缓存和消息中间件。或者说：Redis 是一个高性能的 Key-Value 存储系统，它通过将数据存储在内存中并支持多种数据结构，解决了传统关系型数据库在处理高并发、低延迟场景下的性能瓶颈。 基本概述 分布式存储，在mysql前面 内存型数据库。支持数据库持久化（磁盘） 支持高可用 支持的数据类型 String List Hash Set ZSet Geospatial 地理空间 HyperLoglog 基数统计 核心工具集 redis-server：服务端程序。 redis-cli：命令行客户端。 redis-benchmark：性能压测工具。 redis-check-rdb/aof：持久化文件修复工具。 redis-sentinel：哨兵模式工具。
