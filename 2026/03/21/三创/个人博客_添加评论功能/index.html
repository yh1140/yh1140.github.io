个人博客_添加评论功能

发表于2026-03-21|更新于2026-03-21|博客网站
|浏览量:

方案：Giscus（基于 GitHub Discussions，极简且专业）

这是目前 GitHub 用户最流行的选择。它不需要额外的数据库，所有评论都存储为您仓库里的 Discussions。

1. GitHub 端准备工作

  • 确保仓库公开：您的 yh1140.github.io 必须是 Public。

  • 开启 Discussions：进入您的仓库 Settings -> General -> 找到 Features 模块，勾选 Discussions。

  • 安装 Giscus App：访问 giscus app，点击安装并授权给您的博客仓库。

2. 获取参数

  • 访问 giscus.app。

  • 在“仓库”栏输入：yh1140/yh1140.github.io。

  • 在“分类”栏选择：Announcements。

  • 在页面最下方的“启用 giscus”部分，你会看到生成的脚本。记住其中的 data-repo-id 和 data-category-id。
